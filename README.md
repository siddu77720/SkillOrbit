# SkillOrbit

AI-Powered Skills Matching Platform for career navigation.

## Features
- **Galaxy Map**: 3D interactive job galaxy map.
- **Dream Job Mode**: AI-generated roadmap to your dream job.
- **Skill Score**: Comprehensive grading system based on your skills.
- **AI Assistant**: Conversational agent powered by Groq.
- **Dashboard**: Track your progress and job matching.
- **Project Recommendations**: Build tailored projects to fill skill gaps.

## Setup
1. Clone the repository
2. Run `npm install`
3. Add your Groq API key to `.env` as `VITE_GROQ_API_KEY`
4. Run `npm run dev` to start the app locally.
